# ar-player

 카메라 프레임이 특정 영상이 나타나면 해당 위치에 동영상을 재생하는 기능이다.
 
# 구현할 기능

 1. 기준 영상과 카메라 프레임 사이의 특징점 검출 및 매칭
 2. 호모그래피 계산
 3. 동영상 프레임 투시 변환 및 합성
